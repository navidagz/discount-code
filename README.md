Discount Service
===============

### Possible Improvements
- permission classes
- brand dashboard
    - discount codes list
    - filter on discount fields (user, redeemed, ...)

---

### Run App
```shell
docker-compose up
```
> Documentation URL: [http://localhost:8080/docs](http://localhost:8080/docs)

### Stop App
```shell
docker-compose down
```

### Run tests
```shell
docker-compose run app bash scripts/run_tests.sh
```
